# Mini-Quiz sull'Intelligenza Artificiale

**Gruppo:** Lotito Andrea, Smahi Kawthar, Ouni Yasser

## Descrizione
Mini-quiz interattivo sull'IA realizzato in JavaScript per l'Open Day con i ragazzi delle medie.

## Caratteristiche
✅ **5 domande** sull'Intelligenza Artificiale  
✅ **Domande randomizzate** - ordine diverso ad ogni quiz!  
✅ **Risposte gestite** tramite `alert()` e `prompt()`  
✅ **Messaggi personalizzati** per risposte corrette/errate  
✅ **Interfaccia moderna** e user-friendly  
✅ **Responsive design** per tutti i dispositivi  
✅ **Spiegazioni educative** per ogni domanda  

## Come usare
1. Apri il file `index.html` nel browser
2. Clicca su "Inizia Quiz"
3. Rispondi alle domande digitando 1, 2 o 3
4. Visualizza il tuo punteggio finale!

## File del progetto
- `index.html` - Pagina web principale
- `styles.css` - Stili CSS per l'interfaccia
- `script.js` - Logica JavaScript del quiz
- `README.md` - Documentazione del progetto

## Tecnologie utilizzate
- HTML5
- CSS3 (con animazioni e design responsive)
- JavaScript (ES6+)

## Argomenti del Quiz
- Significato di Intelligenza Artificiale
- Esempi di IA nella vita quotidiana
- Machine Learning
- Applicazioni dell'IA
- Robotica
